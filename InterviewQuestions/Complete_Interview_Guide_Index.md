## 📞 Quick Reference Cards

### Playwright Essentials
```
Auto-wait → Built-in, no explicit waits needed
BrowserContext → Isolated session (like incognito)
Network route → page.route() for mocking
Trace viewer → Debug with screenshots + network logs
Fixtures → Reusable setup/teardown
```

### Selenium Essentials
```
Explicit wait → WebDriverWait + ExpectedConditions
POM → Page Object Model pattern
JavaScriptExecutor → Direct browser control
Selenium Grid → Parallel execution
Exception handling → Try-catch + retry
```
