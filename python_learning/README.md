## 🚀 Getting Started

### Prerequisites
- Python 3.7 or higher
- Jupyter Notebook or JupyterLab

### Installation

1. **Clone or download this workspace**

2. **Create a virtual environment**
   ```bash
   python -m venv venv
   ```

3. **Activate virtual environment**
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

4. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

5. **Start Jupyter**
   ```bash
   jupyter notebook
   ```

6. **Navigate to the python_learning folder and open any module notebook**

## 📖 How to Use

Each notebook module includes:
- **Concepts**: Clear explanations of the topic
- **Code Examples**: Working code snippets with comments
- **Use Cases**: Simple, medium, and difficult level examples
- **Interactive Cells**: Run code directly in the notebook

Start with Module 01 and progress through the modules sequentially for optimal learning. Each module builds on concepts from previous ones.
